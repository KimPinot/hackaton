## Hackaton

```
저는 해커톤을 좋아합니다.
정해진 시간 안에 주어진 목표로 누구보다 빠르게 이동한다는 경험은 회사 업무에서도 쉽게 경험하기 힘들기 때문입니다.

이 프로젝트는 지금까지 여러 해커톤을 진행하며 사용했던 레포지토리들을 이슈 형식으로 아카이브 해두었습니다.

-

I like hackathon.
The experience of moving faster than anyone else to a given goal within a set time is hard to experience even at work.

This project has archived repositories that have been used during several hackathons in the form of issues.
```

## 감상하는 방법 : To appreciate

```
모든 프로젝트는 이슈 형태로 아카이브 되어있습니다.

현재 진행중인 해커톤은 open되어 있고, 이미 종료된 프로젝트는 closed되어 있습니다.

tag에는 진행했던 년도나 결과들과 같은 정보들을 기록해두었습니다.

-

All projects are archived in the form of issues.

The hackathon currently in progress is open, and projects that have already been terminated are closed.

In the tag, we recorded information such as the year and results of the process.
```
